1.2 airlang 语法

词法定义

词法规则

  • 在引号中的字( ‘word’ )代表着这些字符本身。
  • 尖括号( < > )内包含的为必选项。
  • 方括号( [ ] )内包含的为可选项。
  • 大括号( { } )内包含的为可重复0至无数次的项，后接 ‘*’，表示至少一次。
  • 竖线( | )表示在其左右两边任选一项，相当于"OR"的意思。
  • = 是“被定义为”的意思。
